# assets 放置说明

把你的素材放到这里，网站就能引用到。

## 推荐结构
- `public/assets/ps/`：PS 作品图片
- `public/assets/honors/`：荣誉证书图片
- `public/assets/research/`：研究板块图表/流程图（可选）
- `public/assets/avatars/`：头像（可选）
- `public/assets/campus/`：校园作品缩略图（可选）

## 如何在网站里引用
在 `src/content/content.json` 里填写路径，例如：
- `/assets/ps/xxx.jpg`
- `/assets/honors/yyy.png`
- `/assets/research/figure-1.png`
